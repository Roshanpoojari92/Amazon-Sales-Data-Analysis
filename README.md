# 📊 Amazon Sales Data Analysis (Python)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Amazon sales data to identify
revenue trends, profit drivers, demand patterns, pricing behavior, and operational efficiency.

## 🎯 Objectives
- Identify top-performing regions and countries
- Analyze total revenue and profit trends
- Study units sold and pricing patterns
- Evaluate order priority and processing time
- Detect outliers and seasonal trends

## 🧾 Dataset
The dataset contains information related to:
- Order Date, Ship Date
- Region, Country
- Item Type, Sales Channel
- Units Sold, Unit Price, Unit Cost
- Total Revenue, Total Cost, Total Profit

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Analysis Performed
- Revenue & profit analysis by region and country
- Average unit price and cost by item type
- Units sold vs total profit correlation
- Order priority distribution analysis
- Processing time analysis
- Seasonal sales trends
- Outlier detection using IQR method

## 📈 Key Insights
- Certain regions and countries contribute the highest revenue and profit
- Offline sales channel shows higher average unit price
- Positive relationship between units sold and total profit
- Seasonal patterns observed in monthly sales
- Outliers detected in total cost distribution

## ✅ Conclusion
This analysis provides actionable insights that can support strategic decision-making
in pricing, inventory planning, and regional performance optimization.
